What's a variable

Declaring a variable

We use

:=

to declare a new variable.

foo := "bar";

Setting a variable

After it has been declared we use

=

to re-assign it, using

:=

again would reset the declaration.

foo = "foo";

There isn't a typing system

Types are automatically assigned based on context, this brings us to our next lesson.
